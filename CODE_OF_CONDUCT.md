# Code de Conduite - OpenZyra

## Notre engagement

En tant que membres, contributeurs et leaders de la communauté OpenZyra, nous nous engageons à faire de la participation à notre projet une expérience sans harcèlement pour tous, quel que soit l'âge, la taille corporelle, le handicap visible ou invisible, l'origine ethnique, les caractéristiques sexuelles, l'identité et l'expression de genre, le niveau d'expérience, l'éducation, le statut socio-économique, la nationalité, l'apparence personnelle, la race, la religion, ou l'identité et l'orientation sexuelles.

Nous nous engageons à agir et interagir de manière à contribuer à une communauté ouverte, accueillante, diversifiée, inclusive et saine.

## Nos standards

### Comportements positifs

- **Respect** : Faire preuve d'empathie et de bienveillance envers les autres
- **Patience** : Accepter que nous avons tous différents niveaux d'expérience
- **Ouverture d'esprit** : Être réceptif aux suggestions constructives
- **Collaboration** : Mettre l'accent sur ce qui est le mieux pour la communauté
- **Reconnaissance** : Remercier et créditer les contributeurs
- **Communication claire** : Exprimer ses idées de manière constructive

### Comportements inacceptables

Les comportements suivants sont considérés comme du harcèlement et sont inacceptables :

- Utilisation de langage ou d'images sexualisés
- Attaques personnelles, trolling, commentaires insultants/dégradants
- Harcèlement public ou privé
- Publication d'informations privées d'autrui sans permission explicite
- Autre conduite pouvant raisonnablement être considérée comme inappropriée dans un contexte professionnel
- Discrimination sous toutes ses formes

## Responsabilités

### Mainteneurs du projet

Les mainteneurs sont responsables de :
- Clarifier les standards de comportement acceptable
- Prendre des mesures correctives justes et appropriées en réponse aux comportements inacceptables
- Supprimer, modifier ou rejeter les contributions non conformes
- Bannir temporairement ou définitivement les contributeurs pour comportements inappropriés

### Contributeurs

Les contributeurs doivent :
- Respecter ce code de conduite dans tous les espaces du projet
- Signaler les comportements abusifs aux mainteneurs
- Accepter les décisions des mainteneurs concernant les mesures correctives

## Portée d'application

Ce code de conduite s'applique dans tous les espaces du projet, y compris :
- Le repository GitHub (issues, pull requests, discussions)
- Les canaux de communication officiels (email, forum)
- Les événements en ligne ou en personne liés au projet
- Les espaces publics lorsque vous représentez le projet

## Signalement

Si vous êtes témoin ou victime d'un comportement inacceptable :

1. **Contactez-nous** : contact@openzyra.app
2. **Signalement anonyme** : Disponible via notre formulaire de contact

Tous les signalements seront traités avec confidentialité. Les mainteneurs s'engagent à :
- Accuser réception dans les 48h
- Enquêter de manière équitable et rapide
- Prendre des mesures appropriées
- Protéger la confidentialité du signaleur

## Mesures correctives

Les mainteneurs peuvent prendre les mesures suivantes selon la gravité :

1. **Avertissement** : Conversation privée avec avertissement officiel
2. **Exigences** : Demande de correction ou d'excuses publiques
3. **Suspension** : Bannissement temporaire des interactions
4. **Exclusion** : Bannissement permanent du projet
